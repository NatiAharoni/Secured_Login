CommunicationLTD - README

To open the web follow the following steps:

1.1 Open MySQL workbech -> start the server
1.2 Copy 'db_code.sql' from the project folder and run it in MySQL to generate the DB
1.3 Fill your MySQL details in main.py in order to cinnect to the DB

2. Configurate wanted values in config.json (there is an explenation about every variable in password_configuration.py)

3. Run main.py

4. Open the link in the OUTPUT

Note:
If you want your browser to accept the CA certificate follow this tutorial: https://zeropointdevelopment.com/how-to-get-https-working-in-windows-10-localhost-dev-environment/
